Help support Autism awareness by putting a Mothers who have Children with Autism Ribbon on your WordPress site or network.


== Description ==

A small ribbon with a puzzle logo and the words "Proud Mother with an Autistic Child" linking to http://www.facebook.com/pages/Single-Mothers-who-have-Children-with-Autism/258728120832823 Facebook page.
== Installation ==


1. Download the plugin zip archive.

2. Extract it in your `wp-content/plugins` folder or upload via admin panel.

3. Browse to your plugins section and active the plugin.
